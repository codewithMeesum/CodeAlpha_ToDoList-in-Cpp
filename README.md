</p>

<p align="center">
A <b>console-based task manager</b> written in <b>C++</b> that allows users to manage tasks efficiently with persistent storage using file handling.
</p>

---

# ✨ Features

🚀 **Add Tasks**
📋 **View All Tasks**
✅ **Mark Tasks as Completed**
💾 **Save Tasks Automatically (File Storage)**
📂 **Load Tasks When Program Starts**
🧩 **Modular Function-Based Architecture**

---

# 🧠 Concepts Demonstrated

| Concept        | Implementation     |
| -------------- | ------------------ |
| OOP            | Task class         |
| STL            | `vector<Task>`     |
| File Handling  | `fstream`          |
| Modular Code   | Multiple functions |
| Input Handling | `getline()`        |
