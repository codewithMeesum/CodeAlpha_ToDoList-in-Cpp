A simple console-based To-Do List application built using C++.

Features

Add new tasks

View all tasks

Mark tasks as completed

Persistent storage using file handling

Technologies Used

C++

STL Vector

File Handling (fstream)

How to Run

Compile the program:

g++ main.cpp -o todo

Run the program:

./todo

File Structure

main.cpp – Main program
tasks.txt – Stores tasks data
